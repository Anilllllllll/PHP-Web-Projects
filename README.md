## 🚀 How to Run PHP Projects

1. Install **XAMPP / WAMP / MAMP**
2. Move project folder into:

```
htdocs/
```

3. Start **Apache** and **MySQL**
4. Open browser and run:

```
http://localhost/project-folder-name
```
